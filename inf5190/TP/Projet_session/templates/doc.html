/api/demande post

post

Réalisez une demande d'inspection à la ville de Montréal.

post /api/demande

Réalisez une demande d'inspection à la ville de Montréal.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • nom_etablissement: required(string)

    Nom d'établissement

  • adresse: required(string)

    Adresse de l'établissement

  • ville: required(string)

    ville où se situe l'établissement

  • date_visite: required(string)

    Date de la visite du client

  • nom: required(string)

    Nom du client

  • prenom: required(string)

    Prénom du client

  • description_probleme: required(string)

    Description du problème observé

Example:

{
  "nom_etablissement": "BURGER KING",
  "adresse": "203 Victoria St",
  "ville": "Montréal",
  "date_visite": "2023-01-01",
  "nom": "Monsieur",
  "prenom": "Propre",
  "description_probleme": "Il y a une pésence de rats."
}

HTTP status code 201

La demande a été correctement envoyée.

Body

Media type: application/json

Type: any

Example:

{
 "adresse": "800 Victoria Street",
 "date_visite": "2023-08-30",
 "description_probleme": "Il y a des rats",
 "id": 2,
 "nom": "Gaëtan",
 "nom_etablissement": "Macdo",
 "prenom": "Oueyeya",
 "ville": "Longueil"
}

HTTP status code 400

La requête n'est pas correctement écrite.

/api/demande/{id} delete

  • Request
  • Response

  • id: required(string)

HTTP status code 200

/api/utilisateur post

post

Créez vous un compte utilisateur.

post /api/utilisateur

Créez vous un compte utilisateur.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • nom_complet: required(string)

    Nom et prénom de l'utilisateur

  • email: required(string)

    Email de l'utilisateur

  • etablissements: required(array of any)

    Liste des établissements à surveiller

  • password: required(string)

    Mot de passe de l'utilisateur

Example:

{
  "nom_complet": "John Doe",
  "email": "jdoe@gmail.com",
  "etablissements": [
    "La Banquise",
    "St Viateur Bagel"
  ],
  "password": "test123"
}

HTTP status code 201

Le compte utilisateur a correctement été créé.

HTTP status code 409

L'email entré existe déja.
